Instrucciones para obtener repositorio desde GitHub :
1. En el directorio en que desea descargar el proyecto ejecute el comando : git clone https://github.com/johannolayaherrera/Tarea2.git

Intrucciones de ejecución :

Despúes de obtener la información desde el repositorio

1. Ejecute desde el directorio del proyecto el comando : mvn package
2. Luego el comando : mvn exec:java -Dexec.mainClass="com.uniandes.ecos.mavenconteo.Controlador.ControladorConteo" 
3. En la interfaz ingrese el directorio donde se encuentra el código fuente de su proyecto y de clic en el botón calcular.

Nota : Los artefactos del proyecto se encuentran en la siguiente ruta desde el proyecto
\MavenConteo\src\main\resources\PSP1
